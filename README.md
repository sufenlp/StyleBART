# Quick Start
## Data

Please download the data from  [link](), and save it to the `data` folder. 

## Preprocessing

Following the code from [paraphrase](), and then run the code

```
sh paraphrase.sh
```

## Training

```
sh trian.sh
```

## Evaluating

```
sh evaluate.sh
```
